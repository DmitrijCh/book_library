# Book Library

## Описание

Book Library - веб-приложение для управления коллекцией книг. Приложение предоставляет удобный интерфейс для добавления,
поиска, редактирования и удаления книг. Каждая книга содержит информацию о заголовке, авторе и годе публикации.

## Основные функции

1. **Получение списка книг:** Просмотр списка всех книг в библиотеке.
2. **Добавление новой книги:** Добавление новой книги в библиотеку с указанием заголовка, автора и года публикации.
3. **Поиск книг по автору:** Возможность поиска книг по имени автора.
4. **Поиск книг по названию:** Поиск книг по заголовку.
5. **Поиск книг по году публикации:** Поиск книг, опубликованных после указанного года.
6. **Сортировка книг по году:** Получение списка книг, отсортированных по году публикации.
7. **Логирование информации о каждой книге:** Получение подробной информации о каждой книге в виде лога.

## Технологии

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Swagger для документации API
